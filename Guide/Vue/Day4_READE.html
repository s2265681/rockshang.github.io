# 深入令牌机制

前端 Bearer Token 规范

  • 概念：描述在 HTTP 访问 OAuth2 保护自愿时如何使用令牌的规范
  • 特点：令牌就是身份证明，无需证明令牌的所有权
  • 具体规定：在请求头中定义 Authorization:Bear < token >

json Web Token

  • 概念：令牌具体定义方式
  • 规定：令牌由三部分构成“头.载荷.签名”
  • 头：包含加密算法、令牌类型等信息
  • 载荷：包含用户信息、签发时间和过期时间等信息
  • 签名：根据头、载荷及迷药加密得到的哈希串 Hmac Shal 256
  • token:加密防篡改
